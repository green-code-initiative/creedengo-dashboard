str.join or "".join tend to be faster than string concatenation with +

Example

Avoid:

    words = ["this", "is", "a", "list",
         "of", "strings"]

    s = ""
    for word in words:
        s += word

Prefer:

    words = ["this", "is", "a", "list",
         "of", "strings"]

    s = "".join(words)
