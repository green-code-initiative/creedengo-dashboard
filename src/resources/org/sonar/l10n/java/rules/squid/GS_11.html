Ternary operator, while pleasingly compact, can lead to a more difficult to read code. Better to avoid, in favor of the more verbose "if/else" structure.

Example

Avoid:

		String output = i > 10 ? "Yes" : "No";

Opt for:

		String output;
		if(i > 10) {
			output = "Yes";
		} else {
			output = "No";
		}
