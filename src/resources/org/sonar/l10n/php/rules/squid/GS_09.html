The complexity of an expression is defined by the number of &&, || operators it contains. A single expression's complexity should not become too high to keep the code readable.

Example

Avoid:

		if (condition1 && condition2 && condition3 && condition4) { /* ... */ }

Prefer:

		if (relevantMethodName1() && relevantMethodName2()) { /* ... */ }

        /* ... */

        private boolean relevantMethodName1() {
          return condition1 && condition2;
        }

        private boolean relevantMethodName2() {
          return condition3 && condition4;
        }
